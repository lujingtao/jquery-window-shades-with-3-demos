# 一、前言
对于轮播图切换，其实可以有更多效果，其中一类型就是百叶窗。本人参考了网上一些特效，做一些练习和实践，没有封装函数，考虑扩展到SuperSlide。重点是实现效果的思路，并不是代码，掌握了思路可以扩展无限种效果。
